Evidence: content_script.js:230-238, diagnostics/run_mock_full_test.mjs:204-214.

Accepted Residual Risk

P3 Accepted: Local operator token stored in the project folder

The token in backend-proxy/es-backend-token.txt is required for the private local extension/dashboard bridge. It is not the OpenAI key, but the project zip should not be shared publicly with that token intact.

Evidence: service_worker.js:764-771, creator-copilot-service/start-local-operator.ps1:126-135, creator-copilot-service/src/app.js:348-354.

Controls Verified

  • Manifest host permissions are StripChat-scoped with OpenAI and localhost as optional permissions.
  • Extension CSP blocks remote extension-page scripts and eval-style execution.
  • Operator APIs require x-operator-key.
  • Request bodies are size-limited.
  • Webhook signatures use HMAC over the raw body.
  • UI-facing settings strip API keys before returning state.
  • Dashboard renders generated content through textContent.
  • OpenAI output is schema-constrained and exact-count validated.
